# 🍔 Food Delivery Optimizer

A **C++ based route optimization system** that calculates the **shortest
delivery path between locations using Dijkstra's Algorithm**. The
project demonstrates how **graph algorithms can be applied to real‑world
delivery systems** to improve efficiency and reduce delivery time.

------------------------------------------------------------------------

## 🚀 Features

-   Calculates **shortest path between delivery locations**
-   Implements **Dijkstra's Algorithm**
-   Uses **graph data structures**
-   Accepts **user input for nodes and edges**
-   Displays **optimized delivery route**
-   Efficient for **route planning and logistics simulation**

------------------------------------------------------------------------

## 🧠 Algorithm Used

### Dijkstra's Algorithm

This project uses **Dijkstra's shortest path algorithm** to determine
the most efficient route between nodes in a weighted graph.

It works by: 1. Assigning initial distances to all nodes 2. Selecting
the node with the smallest distance 3. Updating distances of neighboring
nodes 4. Repeating until the destination is reached

This ensures the **minimum cost path** is always selected.
